# Fonts Directory

Place your font files (.ttf) in this folder.

## For Aptos Font (PDF Generation)

The application uses the following font files for PDF reports:

| File Name | Usage |
|-----------|-------|
| `Aptos.ttf` | Regular text |
| `Aptos-Bold.ttf` | Bold text (headers, titles) |

## Available Aptos Variants

From the Aptos font family, you may have these variants:
- `Aptos.ttf` - Regular
- `Aptos-Bold.ttf` - Bold
- `Aptos-Italic.ttf` - Italic
- `Aptos-BoldItalic.ttf` - Bold Italic
- `Aptos-Light.ttf` - Light
- `Aptos-SemiBold.ttf` - Semi Bold
- `Aptos-ExtraBold.ttf` - Extra Bold
- `Aptos-Display.ttf` - Display variant
- `Aptos-Narrow.ttf` - Narrow variant

**Required for this app:** Only `Aptos.ttf` (regular) and `Aptos-Bold.ttf` (bold) are needed.

## Instructions

1. Copy `Aptos.ttf` and `Aptos-Bold.ttf` to this folder
2. The app will automatically use them for PDF generation
